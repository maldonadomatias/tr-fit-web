# 004 — 5-day women, hypertrophy, 60min gym (+15-30min cardio)

- **Source**: `APP RUTINA 5 DIAS MUJER 1 HS 15-30 MIN.xlsx`
- **Profile**: mujer, hipertrofia, **5 days**, 60min gym, gym_completo.
- **Purpose**: does the split scale past 4 days, and how is volume distributed?

## The 5-day split decoded

| Day | Emphasis | Block |
|-----|----------|-------|
| D1 | **Quads** + Back | Sentadilla 6a8 / Jalón 6a8 + abs |
| D2 | **Chest** + Shoulders + Triceps (push) | Press Plano 6a8 / Press Militar 6a8 |
| D3 | **Glutes** + Abs | Hip Thrust 6a8 + add/abd + 3× abs |
| D4 | **Back** + Shoulders + Biceps (pull) | Jalón 6a8 / Press Militar 6a8 + curls |
| D5 | **Hams/posterior** + Calves | Peso Muerto 6a8 + Hack + curl femoral |

**Lower body gets 3 days** (D1 quad, D3 glute, D5 ham), upper gets 2 (push D2,
pull D4). This is the **lower-biased** women signature, amplified: ~60% of the
week is legs/glutes, split by sub-region.

## Gate results
- **P1 confirmed**: 5 days = split, lower-biased. Threshold ≤3 full-body / ≥4
  split holds.
- **P2 confirmed**: only D1 has a 2nd warmup (quad→back transition); D2-D5 single
  warmup (one region cluster). Matches "warmup = transitions + 1".
- **P3-P8 confirmed**: identical scheme/RIR/rest/comentario rules.

## New
- **N8 — Lower-body day count scales with frequency** (women):
  3 days → all full-body w/ rotating lower emphasis;
  4 days → 2 lower days (quad / glute-ham);
  5 days → 3 lower days (quad / glute / ham).
  Upper stays at ~2 days (push + pull) from 4 onward.
- **N9 — New rep notations**: `10 C/L` (10 cada lado, unilateral core),
  `30 seg` (plank time-only). Extend the time/laterality scheme family.
- **N10 — A whole day can be glute+abs** (D3) with no upper work — women glute
  priority day.
