## The 4-day split decoded

| Day | Pairing | Lower/Upper |
|-----|---------|-------------|
| D1 | **Quads** + Shoulders | lower(quad) + push-shoulder |
| D2 | **Chest** + Biceps + Triceps | push + arms |
| D3 | **Back** + Shoulders (rear/side) | pull + shoulder |
| D4 | **Glutes/Hams** + Biceps | lower(posterior) + arms |

- Lower body is **split across two days**: quad-emphasis (D1) and
  posterior-chain/glute-emphasis (D4) — classic women's glute priority.
- This is **NOT** classic Push/Pull/Legs. It's a **region-pairing / weak-point
  split**: each day = 1 big region + a complementary smaller region. Shoulders
  appear twice (D1, D3); biceps twice (D2, D4).
- The current generator's templates (Push/biceps, Pull/triceps, Legs+shoulders)
  are a *3-day* PPL — they don't describe this 4-day shape at all.
