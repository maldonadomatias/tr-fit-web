## Split = Quads / Push / Pull / Hams / Push (3 upper / 2 lower)
| Day | Block |
|-----|-------|
| D1 | **Quads** (Sentadilla 6a8) + Pantorrilla + Biceps |
| D2 | **Push**: Pecho (Press Plano 6a8) + Triceps |
| D3 | **Pull**: Espalda (Jalón 6a8) + Hombros (Press Militar 6a8) + Antebrazos |
| D4 | **Hams** (Peso Muerto 6a8) + Aductores + Pantorrilla + Biceps |
| D5 | **Push2**: Pecho Superior (6) + Hombros + Triceps |

- 2 leg days (quad D1, ham D4) → 3 upper / 2 lower. Chest/delts emphasis (push 2×).
- 2 warmups on D1 & D4. 9 slots/day. M1-M5 identical (`fallo`, `40 seg` recur).
