# 005 — 5-day men, hypertrophy, 1 leg day, 60min gym (+30min cardio)

- **Source**: `APP RUTINA 5 DIAS HOMBRE 1 DIA DE PIERNA - 1 HS 30 MIN.xlsx`

## Split = Push / Pull / Legs / Shoulders+Chest / Pull+Arms (upper-DOMINANT)
| Day | Block |
|-----|-------|
| D1 | **Push**: Pecho (Press Plano 6a8) + Hombros (Press Militar 6a8) + Triceps |
| D2 | **Pull**: Espalda (Remo Barra 6a8) + Biceps |
| D3 | **Legs** (Sentadilla 6a8) + Femoral + Pantorrilla + abs + **Cardio Bici 15min** |
| D4 | **Shoulders/Chest**: Press Militar 6a8 + Pecho Superior + rear delts |
| D5 | **Pull/Arms**: Espalda (Jalón 6a8) + Biceps + Triceps |

- 1 leg day → **4 upper / 1 leg = upper-dominant**. Back & arms hit twice (D2,D5).
- 9-10 slots/day. M1-M5 identical.
